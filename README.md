# 海外销售考核Agent

基于TCI海外销售考核规则文档的智能问答与奖金计算工具。

## 功能特性

### 💬 规则问答
- 基于TCI销售考核规则文档
- 支持查询10大类考核规则
- 智能匹配用户问题,提供准确答案

支持的规则类别:
- 综合完成率计算
- 收入完成率计算
- 综合考核分数
- 薪酬结构/固浮比
- 奖金发放时间
- 起付线规则
- 北美奖金计算
- 试用期规则
- 离职结算规则
- 产假规则

### 💰 奖金计算器
- 支持两大地区:北美、非北美
- 支持三种销售角色:直客销售、渠道销售、销售基干
- 基于月薪基数和固浮比自动计算销售奖金
- 实时显示计算结果和详细分解
- 自动检测起付线

## 计算逻辑

### 🇺🇸 北美季度奖金
```
季度基数 = 年薪 × 浮动比例 / 4
销售奖金 = 季度基数 × 奖金系数
```
- 固浮比: 直客50:50, 渠道70:30
- 三段式计算:
  - 51%-100%: 系数1
  - 101%-160%: 超过100%部分系数2.5
  - 161%-300%: 超过160%部分系数1
- 起付线: 51%

### 🌍 非北美年度奖金
```
销售奖金 = 年薪 × 浮动比例 × 综合完成率
```
- 固浮比: 直客60:40, 渠道70:30
- 起付线: 个人无,基干60%

## 使用方法

1. 打开 `overseas-sales-assessment-agent.html` 文件
2. 切换到"规则问答"或"奖金计算器"标签
3. 根据需要输入参数或提问
4. 查看实时计算结果或规则说明

## 技术栈

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

## 文档来源

所有规则基于《TCI 销售考核与奖金计算规则(CN+EN)-202603.docx》

## 注意事项

- 完成率上限: 300%
- 所有计算逻辑严格遵循原始考核规则文档
- 薪酬结构根据地区和角色自动应用对应的固浮比
